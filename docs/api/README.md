# Companion API Documentation

> 本目录包含所有公共接口的文档。

## 接口清单

参见 [SPEC.md](../SPEC.md#11-接口开放清单)。

| 接口 | 类型 | 说明 |
|------|------|------|
| `companion.tool.v1` | MCP | 工具开发接口 |
| `companion.asr.v1` | Rust trait | ASR 实现接口 |
| `companion.tts.v1` | Rust trait | TTS 实现接口 |
| `companion.emotion.v1` | Rust trait | 情绪识别实现接口 |
| `companion.character.v1` | WebSocket Events | 自定义形象前端接口 |
| `companion.prompt.v1` | Config | 对话风格模板接口 |
